考前重點速記

GRC
治理 Governance＋風險 Risk＋法遵 Compliance。目的：達成目標、降低不確定性、確保合規。
公司治理五原則
股 → 董 → 監 → 利 → 資：保障股東權益、強化董事會、發揮監察人、尊重利害關係人、提升資訊透明度。
ERM 五面向
治 → 策 → 營 → 法 → 報：治理、策略、營運、法規、報告。
風險管理四要素
策 → 組 → 流 → 資：管理策略面、風險組織面、風險流程面、風險管理資訊面。
內控三目標
營 → 報 → 法：營運效率及效果、報導可靠及時透明、遵循法令規章。
COSO 五要素
環 → 風 → 控 → 資 → 監：控制環境、風險評估、控制作業、資訊與溝通、監督作業。
內控五配套
授 → 分 → 存 → 驗 → 紀：授權、職能分工、存取控制、獨立驗證、會計紀錄。
三道防線
做 → 管 → 查：第一線營運、第二線風控/法遵、第三線內部稽核。
內部稽核定義
獨立、客觀、確認與諮詢服務；增加價值、改善營運；評估風險管理、控制與治理。
報告關係
功能性報告 → 董事會 / 審計委員會；行政性報告 → CEO / 執行長。
獨立性 vs 客觀性
獨立性＝位置與報告關係，不受干涉；客觀性＝態度公正，不偏不倚。
專業應有注意
合理謹慎，不是保證完全無錯；考量範圍、複雜重要性、內控有效、重大錯誤舞弊、成本效益。
